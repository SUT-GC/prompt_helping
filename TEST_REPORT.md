## 🔧 手动测试指南

虽然自动化测试已覆盖所有代码逻辑，但以下场景建议在真实浏览器中手动测试：

### 手动测试步骤

1. **访问设置页**
   ```
   打开浏览器 → http://localhost:8088 → 点击侧边栏"设置"
   ```

2. **测试配置保存**
   ```
   1. 填写 Base URL: https://ark.cn-beijing.volces.com/api/v3
   2. 填写模型名称：doubao-seed-1-6-251015
   3. 填写有效的 API Key
   4. 点击"保存并验证"按钮
   5. 验证状态指示器切换到"验证中..."
   6. 验证成功后显示"已验证"状态
   ```

3. **测试 API 验证失败**
   ```
   1. 填写无效的 API Key
   2. 点击"保存并验证"按钮
   3. 验证状态指示器切换到"验证失败"
   4. 验证错误 Toast 提示显示
   ```

4. **测试状态指示器同步**
   ```
   1. 查看侧边栏设置按钮旁的状态点
   2. 未配置时应为灰色
   3. 已验证时应为绿色并发光
   ```

5. **测试配置持久化**
   ```
   1. 保存有效配置
   2. 刷新页面
   3. 验证所有输入框自动填充之前保存的值
   4. 验证状态指示器保持"已验证"状态
   ```

6. **测试 API Key 可见性切换**
   ```
   1. 输入 API Key
   2. 点击 👁️ 按钮
   3. 验证 API Key 从密码模式切换到明文模式
   4. 再次点击切换回密码模式
   ```
